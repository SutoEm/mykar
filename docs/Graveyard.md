# Graveyard

This file protects the product from premature scope.

Items here are not bad ideas. They are intentionally out of scope until a later phase review moves them back into the operating system.

## Not Now

- Social media features in year one
- Microservices before the monolith proves pressure
- Generic AI chat before vehicle memory intelligence
- Voice analysis
- Marketplace
- Advertising
- Crypto
- Complex DDD, CQRS, event bus, or repository abstractions
- Direct mobile-to-database product architecture

## Decision Rule

If a feature does not directly strengthen the current phase goal, move it here or to a later phase.
